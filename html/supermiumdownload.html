Up to date browser for outdated version of windows Supermium is a free, open-source browser based on the Chromium browser. A feature of the browser is support for older versions of the Microsoft Windows operating system, starting with Windows XP SP1 of the user line and Windows Server 2003 SP1 of the server line. Windows 2000 support is planned to be added in the future. There are 32-bit and 64-bit versions of the browser. The browser supports logging in to a Google account and synchronizing settings and extensions with Google Chrome.
